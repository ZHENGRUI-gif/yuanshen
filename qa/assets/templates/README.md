# Templates / 模板图片

本目录放 UI 识别用的模板图片（供 AutoHotkey `ImageSearch` 或其他工具使用）。
This folder stores UI template images for matching (used by AutoHotkey `ImageSearch` or other tools).

## Naming / 命名
- Keep filenames stable because scripts reference them.
- 文件名要稳定，因为脚本会直接引用。

## How To Capture / 如何截取
- Capture small, distinctive UI regions (buttons/icons) with minimal background.
- 截取小而有辨识度的区域（按钮/图标），尽量减少背景干扰。

## Troubleshooting / 排查
- If matching fails, try higher-contrast templates, avoid animated parts, and ensure your screen scaling (DPI) is consistent.
- 若匹配失败，优先换更清晰/对比度更高的模板，避开动画区域，并确保 DPI 缩放一致。
